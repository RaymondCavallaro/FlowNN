# Licoes

[English](../../lessons/index.md)

Licoes mantem erros, mecanismos que falharam e mudancas de direcao fora da descricao do sistema atual.

Use esta secao para:

- mecanismos abandonados;
- experimentos que falharam;
- razoes para uma mudanca de design;
- contexto historico que ainda protege o trabalho futuro de repetir um erro.

## Notas atuais de licao

- [Flooding de saida](output-flooding.md)

## Regra

Se uma nota diz "tentamos X e falhou", ela pertence aqui, a menos que seja necessaria para explicar diretamente o comportamento atual do runtime.
