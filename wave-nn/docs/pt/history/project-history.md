# Historico do projeto

[English](../../history/project-history.md)

Esta pagina registra a linhagem do projeto sem fazer branches antigos parecerem arquitetura atual.

## Prateleiras de branch

- `v0.0.1`: fatia de roteamento temporal.
- `v0.0.2`: checkpoint historico, nao uma arquitetura permanente.
- `v0.0.3`: prateleira de campo esparso e consolidacao; sua matematica agora foi promovida para [Modelo matematico](../math/index.md).
- `v0.0.4`: prateleira de aritmetica enquanto aritmetica nao é o foco principal.

## Linha atual de convergencia

`main` é a linha de rede de pressao orientada por significado. O trabalho deve convergir para dinamicas compartilhadas de pressao, valvula, node, campo, recrutamento e regulacao.

## Regra de documentacao

Branches antigos podem explicar origem e licoes. Eles nao devem definir o sistema atual a menos que seu mecanismo tenha sido reintroduzido e testado no `main`.
