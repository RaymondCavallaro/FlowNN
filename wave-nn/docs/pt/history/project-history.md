## Prateleiras de desenvolvimento

Trabalhos anteriores exploraram varias fatias:

- roteamento temporal;
- checkpoints historicos de arquitetura;
- matematica de campo esparso e consolidacao;
- aritmetica como dominio de teste separado.

As partes uteis dessas fatias devem ser promovidas para os docs atuais como conceitos, matematica, licoes ou mecanismos autocontidos.

## Linha atual de convergencia

A linha atual é o experimento de rede de pressao orientada por significado. O trabalho deve convergir para dinamicas compartilhadas de pressao, valvula, node, campo, recrutamento e regulacao.

## Regra de documentacao

Trabalho historico pode explicar origem e licoes. Ele nao deve definir o sistema atual a menos que seu mecanismo tenha sido reintroduzido, documentado em termos atuais e testado.
